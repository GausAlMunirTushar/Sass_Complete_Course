## Sass Variable:

*Without Sass Variables in CSS Code*

```css
body {
  background: red;
}
h1 {
  color: red;
}
span {
  color: red;
}
```
**Within Sass Variables in CSS Code**

```css
$primary-color: red;
$bg-warning: orange;

body{
    background: $bg-warning;
}
h1{
    color: $primary-color;
}
span{
    color: $primary-color:
}
```
